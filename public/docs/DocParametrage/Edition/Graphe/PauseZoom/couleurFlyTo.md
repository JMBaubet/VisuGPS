# Couleur FlyTo

Définit la couleur des marqueurs de FlyTo affichés sur le graphe d'édition.

## Description

Ce paramètre permet de personnaliser la couleur utilisée pour représenter les événements de type "FlyTo" (survol) sur le graphe d'avancement.

## Utilisation

- **Type** : Couleur
- **Défaut** : `orange-darken-1`
- **Impact** : Modifie la couleur de remplissage des marqueurs de FlyTo dans le composant `PauseFlytoGraph`.
