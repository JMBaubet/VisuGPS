# Longueur des marqueurs

Définit la longueur (en pixels) des marqueurs de Pause et de FlyTo sur le graphe d'édition.

## Description

Ce paramètre contrôle la taille horizontale des marqueurs représentant les événements Pause et FlyTo sur le graphe. Une valeur plus élevée rend les marqueurs plus larges et potentiellement plus visibles, mais peut encombrer le graphe si les événements sont nombreux et rapprochés.

## Utilisation

- **Type** : Entier (pixels)
- **Défaut** : `12`
- **Min** : `5`
- **Max** : `50`
- **Impact** : Affecte la largeur des rectangles représentant les pauses et les flytos dans le composant `PauseFlytoGraph`.
