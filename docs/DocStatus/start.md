# Bienvenue sur VisuGPS

VisuGPS est une application conçue pour transformer vos traces GPX en expériences visuelles immersives en 3D. Que vous soyez cycliste, randonneur ou amateur de sports de plein air, VisuGPS vous permet de revivre vos parcours avec une précision géographique exceptionnelle.

## Documentation

L'application dispose d'une documentation complète pour vous accompagner dans la prise en main et la configuration avancée.
Vous la trouverez en cliquant sur l'icône <i class="mdi mdi-book-open-page-variant-outline" style="color: #2196F3;"></i> située en haut à droite de la page d'accueil ou dans les différents panneaux.

## Importation de fichiers

Pour commencer, vous devez placer vos fichiers GPX dans votre dossier de **téléchargement**.

> [!TIP]
> Si vos fichiers GPX n'apparaissent pas à partir du dossier de téléchargement, vous pouvez modifier le nom du dossier par défaut. (voir la documention sur l'importation des fichiers..)
