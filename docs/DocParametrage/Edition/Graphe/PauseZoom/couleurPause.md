# Couleur Pause

Définit la couleur des marqueurs de pause affichés sur le graphe d'édition.

## Description

Ce paramètre permet de personnaliser la couleur utilisée pour représenter les événements de type "Pause" sur le graphe d'avancement. Cela permet de les distinguer visuellement des autres éléments.

## Utilisation

- **Type** : Couleur
- **Défaut** : `purple`
- **Impact** : Modifie la couleur de remplissage des marqueurs de pause dans le composant `PauseFlytoGraph`.
