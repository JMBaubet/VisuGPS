# Afficher les distances

## Description

Ce paramètre active ou désactive l'affichage automatique des bornes kilométriques le long de la trace GPX.

## Valeur par défaut

`true` (activé)

## Utilisation

Lorsque ce paramètre est activé, des messages indiquant la distance depuis le départ (km 10, km 20, etc.) sont automatiquement générés et affichés le long de la trace selon l'intervalle défini.

Ces messages sont créés automatiquement lors de la configuration et peuvent être régénérés à tout moment.

## Impact

- **Activé** : Les bornes kilométriques sont affichées selon la configuration
- **Désactivé** : Aucune borne kilométrique n'est affichée
