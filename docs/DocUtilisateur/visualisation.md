# Mode Visualisation

Le mode Visualisation est l'aboutissement de votre travail : c'est ici que l'animation prend vie.

[< Retour à l'accueil](./index.md) | [< Précédent : Édition](./edition_messages.md)

## Lancer l'Animation

Pour lancer une visualisation :
1.  Depuis la liste des circuits (Accueil), cliquez sur l'icône **Œil**.
2.  L'application charge le terrain et la trace.
3.  L'animation démarre automatiquement après une brève introduction.

## Séquence d'Animation

La lecture suit un scénario précis :
1.  **Intro** : Vue d'ensemble de la trace.
2.  **Départ** : Zoom sur le point de départ (km 0).
3.  **Progression** : Le marqueur avance, la caméra suit les mouvements programmés.
4.  **Évènements** : Les pauses et les survols interrompent le flux normal pour montrer des détails.
5.  **Final** : Zoom arrière une fois l'arrivée atteinte.

## Contrôles de Lecture

Pendant l'animation, vous pouvez interagir :

*   **Barre d'Espace** ou **Clic** : Mettre en Pause / Reprendre.
*   **Touche R** : Recommencer depuis le début (Reset).
*   **Touche M** : Afficher/Masquer le menu de contrôle (si disponible).
*   **Timeline** : Glisser le curseur pour sauter à un moment précis.

## Affichage des Informations

L'écran affiche en temps réel :
*   Le kilométrage parcouru.
*   L'altitude actuelle.
*   La vitesse instantanée (simulée ou réelle selon GPX).
*   Les messages textuels programmés.

## Paramètres Liés

La vitesse globale de l'animation peut être influencée par :
*   `vitesse_animation` : Un facteur multiplicateur définissant à quelle vitesse le marqueur parcourt la trace par rapport au temps réel.
