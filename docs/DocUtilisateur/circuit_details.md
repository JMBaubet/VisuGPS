# Les Données d'un Circuit

La liste principale affiche vos parcours sous forme de cartes ("Cards"). Chaque carte regroupe les informations essentielles et les actions disponibles pour ce circuit.

[< Retour à l'accueil](./index.md) | [< Précédent : Filtres et Tris](./filtres_tris.md)

## Informations Affichées

Une carte de circuit présente les données suivantes :

1.  **En-tête** :
    *   **Nom du circuit** : Titre extrait du fichier GPX ou du nom de fichier.
    *   **Ville de départ** : Commune identifiée au point de départ (avec code postal).
2.  **Vignette** :
    *   Aperçu visuel du tracé sur fond de carte.
3.  **Statistiques** :
    *   **Distance** : Kilométrage total.
    *   **Dénivelé** : Dénivelé positif cumulé (D+).
    *   **Altitude Max** : Point culminant (avec son emplacement kilométrique).
4.  **Métadonnées** :
    *   **Traceur** : Nom de l'auteur.
    *   **Date** : Date de l'import ou de la réalisation.

## Indicateurs d'État

Des icônes ou barres de progression peuvent apparaître pour indiquer l'avancement de votre travail d'édition sur ce circuit (ex: Caméra configurée, Évènements ajoutés).

## Actions

Au bas de chaque carte, une barre d'actions vous permet d'interagir avec le circuit :

*   **Éditer** ![edit](https://api.iconify.design/mdi/pencil.svg?width=20) : Ouvre le mode Édition pour préparer l'animation 3D.
*   **Visualiser** ![visualize](https://api.iconify.design/mdi/eye.svg?width=20) : Lance directement la lecture de l'animation.
*   **Supprimer** ![delete](https://api.iconify.design/mdi/delete.svg?color=red&width=20) : Ouvre la boîte de dialogue de suppression. *(Voir [Suppression](./suppression.md))*
