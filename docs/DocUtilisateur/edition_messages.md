# Gestion des Messages

Les messages permettent d'ajouter des informations interactives (bulles de texte) qui apparaissent sur la carte pendant l'animation.

[< Retour aux généralités Édition](./edition_intro.md)

## Navigation et Commandes (Onglet Message)

Cet onglet permet de positionner précisément l'ancrage visuel de vos messages sur le terrain.

### À la Souris
*   **Clic Gauche + Glisser** : Déplacer la carte (Pan) pour choisir l'endroit exact où la bulle du message doit être ancrée.
*   **Clic Droit + Glisser** : Ajuster l'angle de vue.
*   **Molette** : Zoomer / Dézoomer.

### Au Clavier
*   **Flèches** : Les commandes d'avancement et de pitch restent disponibles pour synchroniser le message avec le passage du marqueur sur la trace.

---

## Ajouter un Message

1.  Positionnez le marqueur au kilomètre souhaité via le clavier ou le graphe.
2.  Dans l'onglet **Messages**, cliquez sur **"Ajouter un message"**.
3.  Utilisez le **clic gauche** pour déplacer la carte et cliquez sur l'icône de visée (ou sur le terrain) pour définir l'ancrage.
4.  Saisissez votre texte dans le champ prévu.

---

## Configuration Avancée

Pour chaque message, vous pouvez régler les paramètres suivants :

*   **Texte** : Le contenu de la bulle informative.
*   **Ancrage** : Les coordonnées géographiques où la bulle s'attache au sol.
*   **Visibilité** : Une case à cocher pour masquer temporairement un message sans le supprimer.
*   **Orientation** : Choisir si le message s'affiche plutôt à gauche ou à droite du parcours (configurable dans les paramètres généraux).

### Apparition Progressive (Fade-in)
Les messages peuvent être synchronisés avec d'autres évènements :
*   **Pendant une Pause** : Le message s'affiche et s'estompe progressivement sur toute la durée de la pause.
*   **Au Départ (km 0)** : Le message peut apparaître pendant le survol inaugural du circuit.

---

## Bibliothèque de Messages
Vous pouvez enregistrer vos messages fréquents (ex: "Col de...", "Ravitaillement") dans une bibliothèque pour les réutiliser rapidement sur d'autres circuits.
