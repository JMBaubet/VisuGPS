# Bibliothèque de Messages

La bibliothèque de messages est votre répertoire central de textes prédéfinis. Elle vous permet de gagner du temps lors de la mise en scène de vos traces en réutilisant des messages courants (Cols, ravitos, anecdotes).

[< Retour à la gestion des messages](./edition_messages.md)

---

## Accéder à la Bibliothèque

Depuis l'onglet **Message** du mode Édition, cliquez sur le bouton :
<span style="background-color: rgba(33, 150, 243, 0.1); color: #2196F3; padding: 2px 10px; border-radius: 4px; font-size: 0.85em;">SÉLECTIONNER UN MESSAGE</span>

### Recherche et Tri
La bibliothèque peut contenir de nombreux messages. Pour vous y retrouver :
*   **Filtrer** : Utilisez la barre de recherche en haut de la fenêtre pour filtrer par mots-clés.
*   **Tri automatique** : Les messages sont triés par ordre alphabétique pour une recherche visuelle rapide.

---

## Types de Messages

Les messages sont classés en deux catégories :
1.  **Messages Utilisateur** : Ceux que vous créez vous-même pour vos besoins spécifiques.
2.  **Messages de Production** : Messages livrés avec l'application (ex: noms de sommets génériques). 
    *   *Note : Les messages de production ne peuvent pas être modifiés ou supprimés en mode normal.*

---

## Gérer vos Messages

### Créer un Nouveau Message
1.  Cliquez sur le bouton <span style="color: #2196F3">NOUVEAU MESSAGE</span>.
2.  Saisissez le **Texte** de votre message.
3.  Choisissez une **Couleur de fond** parmi la palette Material Design proposée.
    > [!TIP]
    > VisuGPS adapte automatiquement la couleur du texte (Noir ou Blanc) en fonction de la couleur de fond choisie pour garantir une lisibilité maximale (contraste).
4.  Cliquez sur **Sauvegarder**.

### Modifier ou Supprimer
Utilisez les icônes à droite de chaque ligne :
*   <img src="https://api.iconify.design/mdi/pencil.svg?width=20" style="vertical-align: middle; margin-bottom: 3px;"> **Modifier** : Pour corriger une faute ou changer la couleur.
*   <img src="https://api.iconify.design/mdi/delete.svg?width=20" style="vertical-align: middle; margin-bottom: 3px;"> **Supprimer** : Retire définitivement le message de votre bibliothèque. Une confirmation vous sera demandée.

---

## Utilisation dans une Trace
Une fois le message idéal trouvé, cliquez sur le bouton <span style="color: #2196F3">SÉLECTIONNER</span>. 
La fenêtre se ferme et le message est prêt à être ancré sur votre parcours. (voir [Détails de l'ancrage](./edition_messages.md#ajouter-un-message))
