# Manuel Utilisateur VisuGPS

Bienvenue dans le manuel utilisateur de **VisuGPS**. Cette application vous permet principalement de visualiser vos traces GPX en 3D et de créer des animations de survol impressionnantes.

## Introduction

VisuGPS est conçu pour mettre en valeur vos parcours GPX. L'interface principale est structurée en trois zones distinctes pour faciliter la gestion de votre bibliothèque.

## 1. Présentation de l'Interface Principale

L'écran d'accueil se décompose de haut en bas. Cliquez sur chaque section pour en savoir plus :

### A. La Barre d'Outils
Située sur la zone supérieure, elle contient les indicateurs techniques (connexion, télécommande, mise à jour des communes), le mode de fonctionnement, et les boutons d'actions globales (importation, paramètres, documentation).
Voir la documentation sur **[La Barre d'Outils](./toolbar.md)**.

### B. La Barre de Filtres et Tris
Située sur la zone intermédiaire, elle permet de rechercher, filtrer (par distance, ville, traceur) et de trier vos circuits pour les retrouver facilement.
Voir la documentation sur **[La Barre de Filtres et Tris](./filtres_tris.md)**.

### C. Les Circuits (Liste)
Située sur la zone centrale, elle affiche vos traces sous forme de liste détaillée. Apprenez à décrypter les informations affichées sur une carte circuit.
Voir la documentation sur **[Les Circuits](./circuit_details.md)**. 

## 2. Utilisation de VisuGPS

Accédez au **[Guide d'Exploitation complet](./exploitation.md)** pour découvrir les fonctionnalités de l'application :

*   Contrôle à distance.
